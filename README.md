# Flappy Bird

A Flappy Bird clone built with Python and Pygame. Features smooth bird animation, scrolling clouds, randomized pipes, score tracking, and a high score system.

## Features

- Animated bird with tilt and wing rendering
- Scrolling sky gradient and clouds
- Randomized pipe gaps
- Score and high score display
- Menu and game over screens
- Forgiving hitbox for fair gameplay

## Requirements

- Python 3.x
- Pygame

```
pip install pygame
```

## How to Run

```
python flappy_bird.py
```

## Controls

| Input | Action |
|-------|--------|
| `Space` or Left Click | Flap / Start / Restart |
| `Esc` | Quit |
